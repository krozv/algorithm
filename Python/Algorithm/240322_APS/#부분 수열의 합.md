# [SWEA] 부분 수열의 합

N개의 자연수

최소 1개 이상의 수를 선택하여 그 합이 K가 되는 경우의 수

## Data Structure

- 이진 트리 형식

## Algorithm

- 완전탐색
- dfs

## How to solve
### Constraint
1. 시간 복잡도

    20개 = 2초, 1개 = 0.1초, 천만번
    
    최대 2^20 = 1,048,576 가능


2. 공간 복잡도

### Method
- DFS를 이용하여 해당 숫자를 선택했는 지 여부를 0과 1로 표현
- 선택한 숫자의 합이 K가 되면 탐색 종료 후 경우의 수 +1
