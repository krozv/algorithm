## How to solve

### Method

- 노드 개수(n)이 주어짐
- 유향 그래프 정보가 주어짐
- 바뀐 노드의 수(m)이 주어짐
- 바뀐 노드의 정보에 따라 유향 그래프를 변경함

### Constraint
1. 시간 복잡도
    - O((n+E) * t)
    - test case (t): 1 <= t <= 100
    - 2 <= n <= 500
    - E = n-1
    - 1초 : 1,000,000,000
